IE Tab 总是需要这个额外的程序吗？ 不可以。Chrome 过去支持一种称为 NPAPI（Netscape 插件 API）的技术，该技术允许扩展程序访问本地计算机资源。 IE Tab 曾经使用该功能访问 Internet Explorer 渲染引擎。但是 Chrome 删除了 NPAPI 支持，因此我们被迫实施了一个新的解决方案，而该解决方案需要一个单独的辅助可执行文件。从技术角度来看，唯一真正的变化是我们的代码从 NPAPI .DLL 文件移动到了辅助 .EXE 文件。 3. IE Tab Helper 应用程序安全吗？ 绝对地！ IE Tab 一直使用原生代码，我们一直非常重视我们 2+ 百万用户的安全。因此，您可以放心，此帮助程序可执行文件是安全且值得信赖的。 4. 我是企业客户，如何自动部署IE Tab Helper？ 我们有企业部署选项，请通过 support@ietab.net 与我们联系以了解详细信息。 可以在此处找到 IE Tab 扩展的隐私政策：http://www.ietab.net/privacy-policy
